## Pourquoi certifier ses compétences ?

Dans un marché IT de plus en plus compétitif, les certifications sont un signal clair de compétence vérifiée. Pour les professionnels de la gestion Apple, les certifications Jamf et Apple sont reconnues par les employeurs, les partenaires et les clients comme des preuves de maîtrise réelle — pas seulement de formation.

Ce module regroupe les guides de révision pour les certifications les plus demandées dans l'écosystème Apple Enterprise et Microsoft 365.

## Certifications Jamf Pro

Jamf propose un parcours certifiant structuré en 4 niveaux, chacun correspondant à une formation officielle :

| Formation | Niveau | Format | Prérequis |
|---|---|---|---|
| **Jamf 100 Course** | Débutant | Self-paced (gratuit) | Aucun |
| **Jamf 200 Course** | Intermédiaire | 4 jours (payant) | Jamf 100 recommandé |
| **Jamf 300 Course** | Avancé | 4 jours (payant) | Jamf 200 recommandé |
| **Jamf 400 Course** | Expert | 4 jours challenge-based | Jamf 300 recommandé |

Les badges numériques Credly obtenus sont partageables sur LinkedIn et valables 3 ans. Voir le [catalogue officiel Jamf](https://trainingcatalog.jamf.com/page/fr-fr).

## Certifications Apple

Apple propose deux certifications professionnelles via Pearson VUE :

**Apple Device Support**
- Valide les compétences de support et dépannage des appareils Apple
- Cible : Help Desk, techniciens support, administrateurs niveau 1-2
- Format : examen en ligne ou centre Pearson VUE

**Apple Deployment and Management**
- Valide les compétences de déploiement MDM et de gestion en entreprise
- Cible : administrateurs IT, ingénieurs MDM, architectes systèmes
- Format : examen en ligne ou centre Pearson VUE

Les deux certifications s'examinent séparément. La combinaison des deux donne accès au titre **Apple Certified IT Professional**.

## Certifications Microsoft

| Certification | Nom complet | Niveau | Public |
|---|---|---|---|
| **MD-102** | Microsoft 365 Certified: Endpoint Administrator Associate | Intermédiaire | Admin Intune/endpoint |
| **MS-102** | Microsoft 365 Certified: Enterprise Administrator Expert | Expert | Admin M365 global |
| **SC-300** | Microsoft Certified: Identity and Access Administrator | Intermédiaire | Admin Entra ID/SSO |

Pour les professionnels de la gestion Apple + Microsoft, MD-102 est la certification prioritaire — elle valide l'utilisation d'Intune pour gérer macOS, iOS et Android.

## Parcours de certification recommandés

**Débutant → Admin macOS**
```
Jamf 100 Course → Jamf 200 Course → Apple Deployment and Management
(6 mois de pratique recommandé entre chaque)
```

**Admin macOS → Sécurité**
```
Jamf 200 Course → Jamf 270 Course (Jamf Protect) → Jamf 370 Course (Security Cloud)
```

**Admin Microsoft → Apple**
```
MD-102 (Intune) → Jamf 100 Course → Co-management Jamf + Intune
```

**Expert complet**
```
Jamf 100 → Jamf 200 → Jamf 300 → Jamf 400 + Apple D&M + MD-102
```

## Stratégie d'examen — Conseils généraux

- **Pratiquez** : les certifications Jamf valident des compétences pratiques, pas seulement théoriques
- **Utilisez un trial** : Jamf propose des essais gratuits de Jamf Pro Cloud
- **Révisez les domaines officiels** : chaque certification publie ses domaines d'examen
- **Quiz de préparation** : utilisez les quiz de chaque cours pour évaluer votre niveau
- **Planifiez** : réservez votre examen 2-3 semaines après votre formation pour maintenir la dynamique
