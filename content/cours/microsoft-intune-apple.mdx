## Vue d'ensemble

Ce module couvre la gestion complète des appareils Apple avec Microsoft Intune dans un environnement Microsoft 365 : intégration ABM, ADE, gestion macOS et iOS, conformité, accès conditionnel et Microsoft Defender.

Module 1 (Fondations Apple Enterprise). Notions Microsoft 365 recommandées.
