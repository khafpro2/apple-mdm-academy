## Introduction

Jamf Pro expose deux API : la **Classic API** (XML, `/JSSResource/`) héritée, et la **Jamf Pro API REST** (`/api/`) en JSON. L'API REST est celle à privilégier pour tous les nouveaux développements. Elle permet d'automatiser n'importe quelle action : créer des groupes, déployer des politiques, récupérer l'inventaire, gérer les utilisateurs — tout ce que l'UI fait, l'API le fait aussi.

> **Documentation officielle :** [Jamf Pro API Overview](https://developer.jamf.com/jamf-pro/docs/jamf-pro-api-overview) (developer.jamf.com, v11.29.0)

## Vue d'ensemble — Jamf Pro API

| Élément | Valeur |
|---|---|
| **Base URL** | `/api` |
| **Documentation intégrée** | `/api/doc` (Swagger UI sur votre instance) |
| **Authentification** | Bearer Token |
| **Obtenir un token** | `POST /v1/auth/token` |
| **Prolonger un token** | `POST /v1/auth/keep-alive` |
| **Expiration du token** | **20 minutes** |

### Méthodes HTTP supportées

`GET` · `POST` · `PUT` · `PATCH` · `DELETE`

### Codes de réponse HTTP courants

| Code | Signification |
|---|---|
| **200** | Succès (lecture ou mise à jour) |
| **201** | Ressource créée |
| **204** | Succès sans contenu (suppression) |
| **400** | Requête invalide |
| **401** | Non authentifié (token absent ou expiré) |
| **403** | Accès refusé (privilèges insuffisants) |
| **404** | Ressource introuvable |
| **409** | Conflit (ex. doublon) |
| **500** | Erreur serveur |

Sur votre instance Jamf Pro, ouvrez `https://votre-instance.jamfcloud.com/api/doc` pour explorer tous les endpoints disponibles avec les schémas de requête et réponse.

## Authentification — Bearer Token

```bash
# Obtenir un token (valide 20 minutes)
TOKEN=$(curl -s -X POST \
  -u "api_user:api_password" \
  -H "Content-Type: application/json" \
  "https://votre-instance.jamfcloud.com/api/v1/auth/token" \
  | python3 -c "import sys,json; print(json.load(sys.stdin)['token'])")

echo "Token obtenu : ${TOKEN:0:20}..."

# Vérifier l'expiration du token
curl -s \
  -H "Authorization: Bearer $TOKEN" \
  "https://votre-instance.jamfcloud.com/api/v1/auth" | python3 -m json.tool

# Renouveler (avant expiration)
curl -s -X POST \
  -H "Authorization: Bearer $TOKEN" \
  "https://votre-instance.jamfcloud.com/api/v1/auth/keep-alive"

# Invalider (toujours faire à la fin)
curl -s -X DELETE \
  -H "Authorization: Bearer $TOKEN" \
  "https://votre-instance.jamfcloud.com/api/v1/auth/invalidate-token"
```

## Opérations courantes

### Lister les Mac (pagination)
```bash
BASE_URL="https://votre-instance.jamfcloud.com"

# Page 1, 50 résultats
curl -s \
  -H "Authorization: Bearer $TOKEN" \
  -H "Accept: application/json" \
  "$BASE_URL/api/v1/computers-preview?page=0&page-size=50" \
  | python3 -m json.tool

# Filtrage : Mac sans FileVault
curl -s \
  -H "Authorization: Bearer $TOKEN" \
  "$BASE_URL/api/v1/computers-preview?filter=diskEncryption.bootPartitionEncryptionDetails.partitionFileVault2State%3D%3DNot%20Encrypted"
```

### Récupérer les détails d'un Mac
```bash
# Par ID
curl -s -H "Authorization: Bearer $TOKEN" \
  "$BASE_URL/api/v1/computers-preview/42" | python3 -m json.tool

# Par numéro de série
SERIAL="C02G1234MD6M"
curl -s -H "Authorization: Bearer $TOKEN" \
  "$BASE_URL/api/v1/computers-preview?filter=hardware.serialNumber%3D%3D%22$SERIAL%22" \
  | python3 -c "import sys,json; d=json.load(sys.stdin); print(d['results'][0]['id'] if d['totalCount'] > 0 else 'Not found')"
```

### Envoyer une commande MDM (wipe, lock, etc.)
```bash
# Envoyer "Update Inventory" à un appareil
COMPUTER_ID=42
curl -s -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  "$BASE_URL/api/v1/jamf-management-framework/redeploy/$COMPUTER_ID"

# Lock un Mac (Lost Mode)
curl -s -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"passcode":"123456","lock_message":"Mac perdu. Appelez IT: +33 1 XX XX XX XX"}' \
  "$BASE_URL/api/v2/mdm/commands" 
```

### Créer un groupe statique
```bash
curl -s -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Déploiement Chrome Wave 1",
    "isSmart": false,
    "computers": [
      {"id": 42},
      {"id": 43},
      {"id": 44}
    ]
  }' \
  "$BASE_URL/api/v1/computer-groups"
```

## Script Python : Rapport de conformité

```python
#!/usr/bin/env python3
"""
Rapport de conformité Jamf Pro — API REST
Génère un CSV avec l'état FileVault de tous les Mac
"""

import requests
import json
import csv
import sys
from datetime import datetime

JAMF_URL = "https://votre-instance.jamfcloud.com"
API_USER = "api-readonly"
API_PASS = "votre-mot-de-passe"

def get_token():
    r = requests.post(f"{JAMF_URL}/api/v1/auth/token", auth=(API_USER, API_PASS))
    r.raise_for_status()
    return r.json()["token"]

def get_computers(token, page_size=100):
    headers = {"Authorization": f"Bearer {token}", "Accept": "application/json"}
    computers = []
    page = 0
    
    while True:
        r = requests.get(
            f"{JAMF_URL}/api/v1/computers-preview",
            headers=headers,
            params={"page": page, "page-size": page_size}
        )
        data = r.json()
        computers.extend(data["results"])
        
        if len(computers) >= data["totalCount"]:
            break
        page += 1
    
    return computers

def invalidate_token(token):
    requests.delete(f"{JAMF_URL}/api/v1/auth/invalidate-token",
                   headers={"Authorization": f"Bearer {token}"})

if __name__ == "__main__":
    token = get_token()
    computers = get_computers(token)
    invalidate_token(token)
    
    # Export CSV
    filename = f"jamf_report_{datetime.now().strftime('%Y%m%d')}.csv"
    with open(filename, "w", newline="") as f:
        writer = csv.writer(f)
        writer.writerow(["Name", "Serial", "OS Version", "Last Check-in"])
        for c in computers:
            writer.writerow([
                c.get("name", ""),
                c.get("hardware", {}).get("serialNumber", ""),
                c.get("operatingSystem", {}).get("version", ""),
                c.get("lastContactTime", "")
            ])
    
    print(f"Rapport généré : {filename} ({len(computers)} Mac)")
```

## Bonnes pratiques

- **Jamais de credentials en clair** dans les scripts → utilisez des variables d'environnement ou un vault (1Password CLI, HashiCorp Vault)
- **Invalidez toujours le token** en fin de script (`trap` Bash pour les erreurs)
- **Gérez la pagination** : ne supposez jamais que tous les résultats tiennent dans une seule page
- **Rate limiting** : Jamf limite les appels API — ajoutez `sleep 0.5` entre les appels en masse
- **Loggez toutes les actions** destructives : wipes, modifications de groupes, etc.
