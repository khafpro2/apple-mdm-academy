## Pourquoi ce module ?

macOS est souvent perçu, à tort, comme un système naturellement sécurisé ne nécessitant aucune gestion spécifique. En réalité, les menaces ciblant macOS ont considérablement augmenté ces dernières années, et les exigences de conformité (ISO 27001, SOC 2, RGPD) s'appliquent pleinement aux parcs Apple.

Ce module vous donne les outils pour sécuriser un parc macOS de façon professionnelle et mesurable, en utilisant les solutions Jamf dédiées à la sécurité.

## Ce que vous allez apprendre

- **Déployer** Jamf Protect pour la détection et réponse aux menaces macOS
- **Configurer** Jamf Security Cloud pour une protection réseau Zero Trust
- **Implémenter** FileVault avec gestion des clés de récupération via Jamf
- **Auditer** Gatekeeper et XProtect pour la sécurité applicative
- **Appliquer** les CIS Benchmarks macOS avec Jamf Protect
- **Détecter** et répondre aux incidents de sécurité sur macOS

## Panorama des menaces macOS

Les principales catégories de menaces ciblant macOS en 2024 :

| Catégorie | Exemples | Protection Jamf |
|---|---|---|
| **Adware / PUP** | Adload, Shlayer | Jamf Protect Analytics |
| **Ransomware** | KeRanger, EvilQuest | Behavioral detection |
| **Cryptominers** | XMRig via app piratée | Process monitoring |
| **Spyware** | Silver Sparrow | Jamf Protect + MDM |
| **Supply chain** | Dépendances compromises | Gatekeeper + Notarization |
| **Insider threat** | Exfiltration de données | Jamf Security Cloud |

## CIS Benchmarks macOS — ce que ça signifie

Le Center for Internet Security publie des recommandations de durcissement pour macOS. Jamf Protect permet d'auditer automatiquement la conformité de chaque Mac par rapport à ces benchmarks :

```
CIS macOS Sonoma Benchmark — Niveau 1 (recommandé)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ 1.1 — Vérifier les mises à jour automatiques
✓ 1.2 — Vérifier la mise à jour des données de sécurité
✓ 2.1 — Désactiver AirDrop en entreprise
✗ 2.6 — Activer le pare-feu macOS       ← Non conforme
✗ 3.1 — Audit logs configurés           ← Non conforme
...
Score de conformité : 43/50 contrôles (86%)
```

Jamf Protect génère des rapports de conformité CIS et peut remédier automatiquement certains écarts via des scripts Jamf Pro.
