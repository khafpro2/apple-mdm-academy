## Pourquoi ce module ?

Jamf Pro est la solution MDM de référence pour la gestion des appareils Apple en entreprise. Ce module couvre l'ensemble du cycle d'administration macOS : de l'enrôlement initial jusqu'à la gestion avancée des politiques, en passant par les profils de configuration, les Smart Groups et le Patch Management.

C'est le cœur de la pratique quotidienne d'un administrateur Jamf. La maîtrise de ces outils permet d'automatiser 90% des tâches d'administration et d'assurer une gestion cohérente à l'échelle d'un parc de milliers de Mac.

## Ce que vous allez apprendre

À la fin de ce module, vous serez capable de :

- **Déployer** Jamf Pro et enrôler des Mac via ADE ou script
- **Créer** des Configuration Profiles pour imposer des paramètres système
- **Configurer** des Policies pour exécuter des scripts et installer des logiciels
- **Construire** des Smart Groups pour cibler dynamiquement vos appareils
- **Déployer** Self Service pour permettre aux utilisateurs d'agir en autonomie
- **Gérer** les mises à jour système avec Patch Management

## Jamf Pro — Architecture et concepts fondamentaux

### La communication MDM

Jamf Pro communique avec les appareils macOS via deux canaux :

```
Mac <──── APNs Push ────> Jamf Pro Server
Mac <──── HTTPS ──────> Jamf Pro Server
Mac <──── Jamf Binary ─> /usr/local/bin/jamf
```

**APNs (Apple Push Notification Service)** : déclenche les actions MDM (installation profils, commandes)
**HTTPS** : transfert de données (inventaire, logs, packages)
**Binary Jamf** : exécute les scripts, policies et opérations locales

### Les trois types d'objets Jamf Pro

| Objet | Rôle | Exemple |
|---|---|---|
| **Configuration Profile** | Impose des paramètres système | Désactiver le Bluetooth, configurer le Wi-Fi |
| **Policy** | Exécute des actions | Installer une app, lancer un script |
| **Smart Group** | Cible dynamique d'appareils | "Tous les Mac sans FileVault" |

## Flux d'administration type

```
Achat appareil → ABM/ADE → Enrôlement Jamf Pro
                                    │
                      ┌─────────────┼─────────────┐
                      │             │             │
               Config Profiles   Policies    Smart Groups
               (paramètres)    (actions)    (ciblage)
                      │             │             │
                      └─────────────┴─────────────┘
                                    │
                              Self Service
                         (autonomie utilisateur)
```
