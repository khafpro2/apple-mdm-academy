## Vue d'ensemble

Ce module couvre Jamf Connect, la solution de gestion d'identité cloud pour macOS : intégration Microsoft Entra ID, authentification OIDC, Single Sign-On, synchronisation des mots de passe et architecture Zero Trust.

Module 2 (Administration macOS avec Jamf Pro) et notions Microsoft Entra ID recommandés.
